# ESP32 迷子防止センサー

ESP32を用いて制作した迷子防止センサーです

親機と子機の通信状態を監視し、一定時間通信が途絶えた際には
ブザーで通知することで、子供の見守りを支援することを目的として開発いたしました。

本プロジェクトでは組込みシステムの設計から実装までを
一貫して行い、Wi-Fi通信やI2C通信などの技術を取り入れています。

---

# プロジェクト概要

本システムは、親機と子機で構成される迷子防止センサーです。

親機・子機ともに一定時間通信できない状態になると、ブザーを鳴らし
LCDに通信状態を表示することで、使用者へ異常を知らせます。

実際の利用シーンを想定し、視覚と聴覚の両方で状況を把握できるよう工夫しました。

# 使用技術

- ESP32
- C/C++
- Aruduino
- ESP-NOW通信
- I2C通信
- LCD表示
- RGB LED制御
- ブザー制御

---

# 主な機能

- 親機・子機間のWi-Fi通信
- RSSIによる通信接続の監視
- しきい値による通信切断時のブザー通知
- LCDへの状態表示
- RGB LEDによる状態表示
- ボタン操作(拡張用・未実装)
- I2C通信フリーズ時の強制再起動

# 工夫した点

- 通信状態をリアルタイムで監視し、異常をすぐに検知できるよう設計しました。
- LCD・RGB LED・ブザーを組み合わせることで、使用者が状態を直感的に把握できるよう工夫しました。
- 子供が使用することを前提とし、子機のLCD表示は顔文字を採用、またRGB LEDを使用し状態変化をわかりやすくしました。
- 通信が切断された際、通信範囲内へ戻ると自動で再接続を行い、使用者が再設定をしなくても継続して使用できるようにしました。
- LCDのフリーズ対策として、フリーズを常時監視してフリーズを検知した場合は強制再起動を実装し、安定した動作を目指しました。
- フローチャートやシステム概要を作成し、設計から実装までの流れを整理しました。
